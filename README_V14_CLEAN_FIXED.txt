# الملكي POS v14 Clean Fixed

هذا إصدار مبني من جديد لتصحيح المشاكل:
- لا يظهر تحويل المخزون في المحلات العادية.
- لا تظهر إضافة منتج جديد للمحلات العادية.
- المصنع فقط يستطيع إضافة مخزون للمحلات مع خصم من مخزون المصنع.
- الأدمن لديه Dashboard مختصر فعلي.
- أسماء المحلات كأزرار وتظهر فواتير اليوم مع المنتجات والأسعار.
- تم إصلاح getPendingInvoices و savePendingInvoice و setSession.
- حفظ الفاتورة = فواتير معلقة.
- بيع = بيع نهائي.
- حذف = إخفاء فقط برسالة "آخر قرار ؟".
- إعادة طباعة موجودة.
- تقارير الأدمن أسبوعي/شهري/سنوي/فترة.
- مصاريف الأدمن لحساب الربح.
