# الملكي POS v11 - Pending Invoices and Clients

## الجديد
- زر حفظ الفاتورة أصبح يحفظها في قائمة فواتير معلقة.
- زر جديد "بيع" يقوم بالدور القديم: حفظ الفاتورة النهائية وتحديث المخزون.
- صفحة "فواتير معلقة" لبيع أو إلغاء الفواتير المعلقة.
- حقل العميل ببحث ذكي.
- الافتراضي Customer.
- عند كتابة عميل جديد يتم حفظه في Firebase.
- في التقارير: عمود "حذف أو إعادة طباعة".
- حذف الفاتورة أصبح إخفاء فقط من حساب المحل، ولا تُحذف من قاعدة البيانات.
- الأدمن يرى عدد الفواتير المخفية لكل محل.
- زر إعادة طباعة لكل فاتورة.
