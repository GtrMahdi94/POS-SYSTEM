# الملكي POS v12 Fixed

## التصحيحات
- الموردين وتحويل مخزون بين المحلات يظهران فقط للمصنع.
- حسابات الكاشير أزيلت من صفحات المحلات.
- إحصائيات الأدمن تم إصلاحها.
- إضافة محل المصنع بكلمة مرور 4444.
- بيانات البيع لا تحتوي اختيار أسماء البائعين.
- الجلسة تحفظ بعد Refresh ولا يرجع النظام لشاشة البداية.
- زر حفظ الفاتورة أصبح يحفظ في فواتير معلقة.
- زر بيع أضيف للسلة وينفذ البيع النهائي.
- التقارير: حذف أو إعادة طباعة.
- حذف الفاتورة إخفاء فقط، وليس حذف من Firebase.
- الأدمن يرى عدد الفواتير المخفية.
